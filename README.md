# Student Result Calculator

This is a simple Python program that calculates the total marks, percentage, grade, and status (PASS/FAIL) of a student based on the marks obtained in five subjects.

## Features

- Takes input marks for five subjects.
- Calculates total marks and percentage.
- Assigns a grade based on percentage.
- Determines pass/fail status (must score at least 35 in each subject to pass).
- Displays the final result with all details.

## Usage

1. Ensure you have Python installed on your system.
2. Run the program:
   ```bash
   python student_result_calculator.py
   ```
3. Enter the marks for all five subjects when prompted.

## Note

- Enter numerical values for marks.
- Minimum passing mark for each subject is 35.
